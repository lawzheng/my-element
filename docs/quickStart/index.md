## 快速开始

1. 在项目目录里安装`@lawzz/my-element`
    - npm安装 `npm i @lawzz/my-element`

2. 在`main.ts`中全局引入
```js
import mUI from '@lawzz/my-element'
import '../node_modules/@lawzz/my-element/lib/style.css'

app.use(mUI)
```

3. 按需引入

```js
import chooseIcon from '@lawzz/my-element/lib/chooseIcon'

app.use(chooseIcon)
```

4. 使用
